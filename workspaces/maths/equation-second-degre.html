Equation générale du second degré

Equation générale permettant de trouver les zéros, si ils existent, d'une équation du second degré

Dans l'équation générale du second degré:

ax2+bx+c=0a x^2 + b x + c = 0 ax2+bx+c=0

Si a≠0a\not=0a=0 il existe deux solutions a l'équation ax2+bx+c=0a x^2 + b x + c = 0ax2+bx+c=0 qui sont:

x=−b±b2−4ac2ax = \frac{-b \plusmn \sqrt{b^2-4ac} }{2a} x=2a−b±b2−4ac​​

Nous avons

x1=−b+b2−4ac2ax_1 = \frac{-b + \sqrt{b^2-4ac}}{2a} x1​=2a−b+b2−4ac​​

et

x2=−b−b2−4ac2ax_2 = \frac{-b - \sqrt{b^2-4ac}}{2a} x2​=2a−b−b2−4ac​​

On pose: Δ=b2−4ac\Delta = b^2 - 4 acΔ=b2−4ac qu'on appelle le Discrimant de l'équation

Comme l'équation comprend une racine carrée de Δ\DeltaΔ, dans R:\Reals: R: Δ\sqrt{\Delta}Δ​ n'existe que si Δ\DeltaΔ est null ou positif.

Δ∈R  ⟺  Δ≥0\sqrt{\Delta} \in \Reals \iff \Delta \ge 0 Δ​∈R⟺Δ≥0
